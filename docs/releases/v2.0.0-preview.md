# Vouch Starter Kit v2.0.0-preview

Vouch Starter Kit is now a local-first decision intelligence engine and reference app, not only a CSV dashboard.

## Highlights

- Explore-in-30-seconds completed workspaces
- Stable direct routes for four primary verticals
- Typed declarative packs and Recruitment Agency example
- Headless analysis API
- Review/pack exports and sanitized SVG sharing
- Progressive evidence and developer diagnostics
- Privacy messaging and disabled telemetry
- Expanded contributor, architecture, embedding and deployment documentation
- CI, security and repository contribution workflows

## Upgrade and migration

IndexedDB moves to schema 3 without deleting reviews/actions. Existing workspace mappings/preferences are preserved and a backup recommendation is added. Export a JSON backup before any replace import or origin change.

## Known limitations

No accounts, cloud sync, connectors, scheduled ingestion, collaboration or notifications. Portable pack JSON cannot express executable custom rules. Local data remains tied to one browser origin/profile. Review JSON contains normalized records and is not a public-sharing format.

## Contribute

Start with `.github/GOOD_FIRST_ISSUES.md`, build a pack using `docs/BUILD_A_VERTICAL_PACK.md`, or propose an integration architecture.
