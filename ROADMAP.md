# Roadmap

Vouch is being built incrementally, in public, with a focus on usefulness at every stage. This document describes where we're headed and why.

---

## Vision

A world where any business — regardless of size or technical resources — can understand and act on the data they already have. Vouch aims to be the simplest possible path from raw business data to clear, actionable revenue insights.

---

## Guiding Principles

- **Ship early, ship often** — each milestone should be genuinely useful on its own
- **Don't require infrastructure** — if it needs a data engineer to set up, it's too complex
- **Trust through transparency** — users should be able to understand how every insight was reached
- **Community-driven** — features are prioritized by real-world need, not roadmap theater

---

## Milestones

### v0.1 — Foundation

The first usable version. Focused entirely on making the core workflow work reliably.

- [ ] Upload a CSV file
- [ ] Auto-detect column names and data types
- [ ] Summary dashboard with key metrics
- [ ] Revenue opportunity insights (lapsed customers, anomalies, pricing gaps)
- [ ] Export a summary report

**Goal:** Someone should be able to upload a sales export and get a useful insight in under five minutes.

---

### v0.2 — Intelligence

Making the results smarter and more actionable.

- [ ] Improved charts and visualizations
- [ ] Smarter field mapping with user confirmation step
- [ ] AI-generated recommendations with plain-language reasoning
- [ ] Saved sessions (revisit a previous analysis)

**Goal:** Results should feel tailored to your data, not generic.

---

### v1.0 — Production-Ready Community Edition

A stable, documented release that teams and consultants can rely on.

- [ ] Stable, versioned public API
- [ ] Self-hosted deployment guide (Docker, cloud options)
- [ ] Multi-file and multi-source support
- [ ] Community plugin system for custom insight types
- [ ] Full documentation site

**Goal:** Production-grade reliability that organizations can build on.

---

## Beyond v1.0

Ideas being considered for future releases — not committed, subject to community input:

- Native integrations (Shopify, Stripe, QuickBooks exports)
- Scheduled analysis runs
- Team collaboration features
- Role-based access for self-hosted deployments

---

## How to Influence the Roadmap

This roadmap is a living document. If you have a use case that isn't represented here, open a [GitHub Issue](../../issues) and describe it. The best way to move something up the list is to show that real people need it.
