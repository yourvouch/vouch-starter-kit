# Roadmap

## 2.0 Preview — current

- Local-first review, persistence, comparison, actions and outcomes
- Declarative built-in/example packs
- Headless analysis and portable exports
- Stable sample workspaces and contributor documentation

## Preview hardening

- Validate more real-world CSV/XLSX dialects
- Improve accessible mobile alternatives for dense comparisons
- Expand safe rule DSL operators without executable imports
- Add more community-reviewed pack test fixtures
- Establish a documented npm package boundary; no package is published yet

## Stable 2.0 criteria

- External pack contributions validate cleanly
- Migration and import behavior is exercised across released schemas
- Accessibility and performance budgets are enforced in CI
- Public API compatibility policy is documented

## Commercial boundary

Cloud sync, connectors, scheduled ingestion, team permissions, notifications, automation, audit logs, managed packs and support remain commercial Vouch capabilities. Open-source decision logic will not be artificially restricted.

Influence priorities through GitHub Issues or Discussions.
