# Product specification — Vouch Starter Kit 2.0 Preview

## Purpose

Provide a local-first, open-source decision intelligence engine and reference workflow that turns existing CSV/XLSX business data into explainable priorities, actions, comparisons and associated outcomes.

## Primary question

Every product screen should answer: **What deserves my attention?**

## Users

Operators use the reference review. Developers, consultants, agencies and vertical SaaS builders configure packs, embed the headless engine or deploy branded forks.

## Core guarantees

- no signup/backend for core use
- local file parsing and no source-binary persistence
- deterministic versioned rules
- honest unavailable evidence
- immutable saved reviews
- confidence-aware comparison
- persistent actions and non-causal outcomes
- versioned portable exports and deletion controls

## Product boundary

Open source proves intelligence locally. Commercial Vouch adds managed deployment, cloud synchronization, connectors, collaboration, automation, operational cadence and support.
